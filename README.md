# Calculator

> A small calculator app to practice DOM manipulation, CSS styling, HTML structure, and JS scripts, by creating a writing functions to operate the calculator. Slightly away from the orgional intent of the project to create a simple calculator. This app can use multiple inputs and evaluate according to PEMDAS (or the last 4 at least).

# Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Silas-N-Schlax/rm-prework-odin-calculator.git
```

Navigate to the project directory:

```bash
cd rm-prework-odin-calculator
```

Open `index.html` in your browser.

# Reflection

Key things learned from building this project:

* Key events and adding events to them.
* Actually using switch case for once.
* Understanding more in depth how PEMDAS works to figure out how to tell the computer how to evaluate the expression correctly.

Things I would improve in a future iteration:

* Added features for exponents and "(" ")".
* Update UI and add mobile compatibility.
* Simplify my complex code used to evaluate PEMDAS.
